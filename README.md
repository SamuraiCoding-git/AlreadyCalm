# Already Calm

Premium noosphere / cosmic-spiritual landing page built with Next.js 15, Tailwind CSS, Framer Motion, and a lightweight `@react-three/fiber` ambient scene.

## Components

- `app/page.tsx` - page composition and section order
- `components/landing/hero-section.tsx` - immersive hero with countdown, CTA, and preview card
- `components/landing/trust-bar.tsx` - glowing trust stats strip
- `components/landing/pain-section.tsx` - pain/agitation cards
- `components/landing/core-truth-section.tsx` - aha moment and neural truth list
- `components/landing/reset-section.tsx` - 10-minute reset timeline
- `components/landing/social-proof-section.tsx` - testimonials grid
- `components/landing/offer-section.tsx` - offer, price anchor, and guarantee
- `components/landing/faq-final-section.tsx` - FAQ accordion and final CTA
- `components/landing/sticky-bar.tsx` - sticky mobile/desktop CTA bar with progress line
- `components/landing/exit-popup.tsx` - exit-intent / delayed popup
- `components/landing/cosmic-cursor.tsx` - glowing orb cursor
- `components/landing/ambient-canvas.tsx` - subtle 3D neural particle field
- `components/landing/countdown.tsx` - persistent offer timer
- `components/landing/reveal.tsx` - scroll reveal animation wrapper
- `components/landing/section-heading.tsx` - shared section heading pattern
- `components/landing/footer.tsx` - footer
- `lib/content.ts` - centralized landing copy and content data

## Run

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Production

```bash
npm run build
npm start
```
